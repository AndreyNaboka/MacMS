# MacMS

Нативный монитор CPU и оперативной памяти для строки меню macOS.

## Возможности

- два графика в реальном времени в строке меню: общая загрузка CPU и RAM;
- история загрузки примерно за последние 34 секунды;
- окно со списком процессов, их CPU и резидентной памятью;
- сортировка по любой колонке кликом по заголовку;
- приложение не занимает место в Dock.

## Запуск

Требуются macOS 13 или новее и Xcode/Swift toolchain.

```bash
swift run
```

Также можно открыть `Package.swift` в Xcode, выбрать схему **MacMS** и нажать Run.

После запуска нажмите на графики в строке меню, чтобы открыть список процессов.
Первое измерение CPU процесса равно нулю: для вычисления загрузки нужны две последовательные выборки.
